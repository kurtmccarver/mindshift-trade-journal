# Changelog

## 1.0.0 - 2026-07-08

Initial production release.

- Added SvelteKit trade journal application
- Added home workspace, dashboard, trades, calculator, settings, privacy, terms, and admin pages
- Added simple mode and prop-firm mode
- Added instruction modal on site load
- Added inline editable trade table and custom columns
- Added smart `signal by` custom column normalization and dashboard chart
- Added CSV import and export
- Added supported exchange price lookup and manual price fallback
- Added Supabase-backed feedback modal and private feedback admin page
- Added server-side rate limiting for feedback and price APIs
- Added Vercel production build support
